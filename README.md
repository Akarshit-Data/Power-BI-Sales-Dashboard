# Power-BI-Sales-Dashboard
Interactive Sales Performance Dashboard Using SuperStore Dataset
Project Overview
This project involves the creation of an interactive Sales Dashboard to track $887K in revenue. The goal was to identify seasonal trends, regional performance, and top-selling product categories.

Key Features
Interactive Slicers: Filter by Year to see dynamic data shifts.

Time-Series Analysis: Custom Calendar table to track monthly sales peaks.

KPI Tracking: Real-time calculation of Profit Margin and Total Orders.

Tools Used
Power BI Desktop: Data Visualization & Modeling.

DAX: For calculated measures.

Power Query: For data cleaning and transformation.
